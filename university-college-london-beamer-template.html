University College London Beamer Template

Date Sun 14 July 2013

In this post I’ll briefly go over my UCL template for Beamer presentations.

Why LaTeX?

  • Neat Equation formatting
  • Excellent typesetting
  • pdf files work anywhere

Why Beamer?

  • Images and equations very quickly turned into a presentation
  • Font sizes are always sensible

Why XeTeX?

  • Free choice of any font installed

So to begin we will need to ensure an upto date distribution of LaTeX installed. On ubuntu this is simply:

sudo apt-get install texlive-full

Alternatively head to texlive and follow the appropriate instructions for your system. The template given here builds upon the lovely code from Ed Long. The main difference when it comes to using XeTeX with beamer is we add

\setallmainfonts(Digits,Latin){<FONTNAME>}

to update the main font to our choice. In this case we’re not changing the font mathematics will appear in, purely because most fonts won’t render very nicely. Throughout the template we set the colour of the UCL header and the text colour the same. The first step in setting that colour is

\definecolor{maincolour}{rgb}{<COLOUR>}

Compiling our presentation file uses the following command

xelatex <PRESENTATION>.tex

The link to the actual code is at the bottom of the post. Before then I’ll quickly go over an example. And remember - UCL does technically have a style guide, use to grab some interesting colours (and perhaps ignore their font choice). So to begin grab a copy of the code from : https://github.com/marksbrown/xetextemplate

Designing our slides

This is pretty much choosing a colour and font we like. Preferably both being clear and clean to read. For this example we’ll choose "Norasi" as the font and a lovely olive green. The first thing to be done is updating the colour of the UCL header. With the files in images/svg lies ‘uclheader.svg’. I’d recommend opening this up in Inkscape and changing the colour of both elements to your colour of choice using the ‘fill and stroke’. Following this, save a pdf copy to the images/pdf and voila! Step one done.

Next we need to make a note of the colour we’ve just selected.

For LaTeX we divide the RGB by 255 to give our colour as

\definecolor{maincolour}{rgb}{0,0.47,0.13}

With the font updated to our choice we arrive at our new theme

Enjoy playing around!
